# HEKAX Voice AI

## Real-Time Voice Intelligence System

HEKAX Voice AI is a real-time voice intelligence system designed for low latency, high clarity, and reliable performance in production environments.

---

## Applications

- AI phone receptionist
- Call transcription
- Caller intent extraction
- Response generation
- Audio-based decision making

---

## Capabilities

### Text-to-Speech (TTS)

- Natural, neutral global English
- Low latency output
- Configurable tone and pacing
- Hardware-optimized inference

### Speech-to-Text (STT)

- Real-time transcription
- Segment streaming
- Noise robustness
- Multi-language support

### Auxiliary Voice Intelligence

- Voice activity detection
- Keyword spotting
- Emotion scoring
- Segment-based classification

---

## Training and Pipeline

The system includes:

- Data preprocessing
- Audio segmentation
- Embedding model training
- Diarization and speaker separation
- Evaluation scripts

---

## Intended Usage

Designed for high-call-volume businesses requiring reliability, clarity, and consistent AI-driven call handling.
